## SAL-051 — External-application API: PATs + session-less /api/v1 + MCP (2026-07-18)

**Severity:** Medium (new public surface, by design) · **Status:** Mitigated at design + implementation (foundation PR); pre-GA checklist below tracks the remaining probes.

**Design commitments implemented in `20260718150000_integrations_foundation.sql`:**
- Tokens: `shyre_pat_` + 256-bit CSPRNG, sha256 at rest, prefix-only display, shown once, 90-day default / 1-year max expiry, revoke-only (no delete — forensics), bound to one (user, team) at creation. Team id derives from the token row, never the request.
