# Getting started

A guided tour from first login to first invoice. Works for any account.

## 1. Sign in

Open your Shyre URL and sign in with the account that was set up for you. Shyre uses Supabase Auth; email + password today, MFA optional from user settings.

On first sign-in, a **personal organization** is created automatically. You are its owner. Everything you create lives inside an organization, even if you never invite anyone else.

## 2. The sidebar

- **Track** — Dashboard, Time
- **Manage** — Customers, Projects, Invoices, Reports
- **Admin** — Organizations, Security Groups, Categories, Templates, Import, Business
- **System Admin** — only visible if you are a platform administrator

Your profile lives behind your name/avatar at the top of the sidebar.

## 3. Create your first customer

1. Sidebar → **Customers**
2. Press `N` or click **Add customer**
3. Fill in at minimum a name. Email, default rate, and address are optional but help later.
4. Save.

## 4. Create your first project

1. Sidebar → **Projects**
2. Press `N` or click **New project**
3. Pick the customer you just created (or leave blank for an internal project).
4. Set an hourly rate if the customer doesn't already have one.
5. Save.

## 5. Track time

1. Sidebar → **Time**
2. The top card is your timer. Pick the project, add an optional description, hit **Start** (or press `Space`).
3. Stop the timer with **Stop** (or `Space` again).
4. Past time: click **Add past entry**, or press `N`, to log something you forgot.

The week grid below shows every entry, split by day. Click any entry to edit inline. Kebab menu offers **Edit**, **Duplicate** (start a new timer with the same project + description), **Delete**.

## 6. Send an invoice

1. Sidebar → **Invoices** → **New invoice**
2. Pick the customer. Shyre shows uninvoiced, billable time entries.
3. Review line items, adjust amounts if needed.
4. Save as draft, then send when ready.

Invoices are in early development. Expect basics now; late fees, multi-currency, and recurring invoices are planned.
