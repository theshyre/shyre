# Templates

Templates pre-fill the common shape of a time entry so starting one takes a single click.

## Creating a template

1. Sidebar → **Templates**
2. Click **New template**
3. Set any combination of:
   - Project
   - Description
   - Category (must match the project's category set, if any)
   - Billable flag
4. Name the template (e.g. "Standup", "Code review for Acme").
5. Save.

Templates are scoped per user within an org. Other org members don't see yours.

## Using a template

On the Time page, look for the template chips under the running-timer card. Click a chip to start a timer with those values pre-filled. You can adjust any field after starting.

Under the chips, a kebab menu on each template lets you edit or delete.

## Why templates

If 80% of your tracked time falls into 4–5 repeating patterns, templates eliminate the "pick the project, type the description" tax every single time. Solo consultants especially: one standup template and one "email + admin" template will cover a lot.
