# Stint quick guide

Log your first day of time in under five minutes. This is the fast path —
for the full picture see [Time tracking](time-tracking.md) and the rest of
the Stint topic.

## 1. Open Time

Sidebar → **Time** (or press `T`). This is `/time-entries`, your daily
driver.

## 2. Start a timer

Pick a project (or an [internal project](internal-projects.md) for
unbillable work), type what you're doing, and hit **Start**. No project yet?
Create one inline from the picker.

## 3. Tag it with a category

Categories answer "what" (project already answers "for whom"). Add one from
the entry row — see [Categories](categories.md) if you want to set up your
own set first.

## 4. Stop the timer, or edit the time directly

Click **Stop** when you're done, or skip the timer entirely and type start/
end times straight into a row — both produce the same entry.

## 5. Check the week

Switch to Week view to see the day you just logged next to the rest of the
week. Entries always show who logged them, so this holds true on shared
projects too.
